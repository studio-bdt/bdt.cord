# BDT.cord

BDT.cord is a website built for anonymous chatting. Users can give themselves a display room and then proceed to either browse public rooms to join, or join a private room.

This project is now considered completed, but we will occasionally code new updates. 

Created by <a href="https://studio-bdt.github.io">Studio BDT</a>
